소개

안녕하세요! 웹스토리보이입니다. 첫 번째 레이아웃은 잘 하셨죠? 막상 해보니까 그렇게 어려운건 없죠? 두 번째 레이아웃도 잘 따라하시면 됩니다. 그럼 시작해 볼까요? 이번 레이아웃은 기존 레이아웃에서 하나의 박스가 더 생긴 구조입니다. 구조가 조금 더 복잡해졌기 때문에 contents영역 속에 3개의 박스를 만들겁니다. 레이아웃이 한번 더 복잡해지긴 하지만 어렵지 않습니다.

이론

두번째 layout02.html 파일을 만들고 !를 쓰고 tab을 누르고 4개의 박스를 만들고, 4개의 박스를 감싸는 container를 만들겁니다. 

<!DOCTYPE html>
<html lang="ko">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Layout02</title>
    </head>
    <body>
        <div id="container">
            <div id="header"></div>
            <div id="nav"></div>
            <div id="contents"></div>
            <div id="footer"></div>
        </div>
    </body>
</html>

각각의 태그에 이름을 만들어 줬습니다. 지금처럼 태그를 선택해야 할 경우, 이름이 설정해야 할 경우, 선택을 해야 할 경우, 이런 경우 선택자를 사용하게 됩니다. 가장 기본적인 선택자1로는 아이디 선택자와 클래스 선택자가 있습니다. 우선은 id 선택자를 이용하여 사용할 것입니다. 이번에는 CSS를 설정하겠습니다. 

 * {
    margin: 0;
    padding: 0;
}
body {
    background-color: #E8F5E9;
}
#container {
    width: 1200px;
    margin: 0 auto;
}
#header {
    width: 1200px;
    height: 100px;
    background-color: #C8E6C9;
}
#nav {
    width: 1200px;
    height: 100px;
    background-color: #A5D6A7;
}
#contents {
    width: 1200px;
    height: 780px;
    background-color: #81C784;
}
#footer {
    width: 1200px;
    height: 100px;
    background-color: #43A047;
}
